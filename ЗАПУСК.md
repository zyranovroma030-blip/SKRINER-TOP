# Как запустить скринер

## Запуск в один клик (рекомендуется)

Дважды щёлкните по файлу **start.bat** в корне проекта. Будут установлены зависимости (если нужно), запустятся сервер и клиент, откроется браузер на http://localhost:5173.

---

## Вариант 1: Только интерфейс (проще всего)

1. Откройте папку проекта в терминале (PowerShell или cmd).
2. Выполните по очереди:

```bash
npm install
npm install --prefix client
```

3. Запустите приложение:

```bash
npm run start
```

Или из папки **client**:

```bash
cd client
npm install
npm run dev
```

4. Откройте в браузере: **http://localhost:5173**

---

## Вариант 2: Всё одной командой из корня

В корне проекта (где лежит этот файл) выполните:

```bash
npm run install:all
npm run start
```

После этого откройте **http://localhost:5173**.

---

## Вариант 3: С оповещениями в Telegram

1. Установите зависимости (из корня проекта):

```bash
npm run install:all
```

2. В папке **server** создайте файл **.env** и добавьте строку (токен от @BotFather):

```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
```

3. Запустите клиент и сервер в **двух разных окнах терминала**:

**Окно 1 — интерфейс:**
```bash
npm run start
```

**Окно 2 — сервер для Telegram:**
```bash
npm run dev:server
```

4. В браузере откройте http://localhost:5173, вкладка «Оповещения» — введите Telegram Chat ID и создайте оповещения.

---

## Если не запускается

- **«npm не является внутренней или внешней командой»** — установите Node.js с https://nodejs.org (версия 18 или новее). Перезапустите терминал после установки.
- **Ошибка при `npm install`** — откройте терминал **от имени администратора** или запустите из папки без кириллицы (например, скопируйте проект в `C:\projects\bybit-screener` и откройте эту папку в терминале).
- **Порт 5173 занят** — закройте другое приложение на этом порту или в файле **client/vite.config.ts** измените `port: 5173` на другой (например, `port: 3000`).
