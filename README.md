# Bybit Screener

Скринер фьючерсов Bybit в стиле Digash: графики, фильтры по объёму и волатильности, карта плотностей по стакану, оповещения в Telegram. Без подписки.

## Запуск

**Быстрый старт (только интерфейс):**

```bash
npm run install:all
npm run start
```

Откройте в браузере: **http://localhost:5173**

Подробная инструкция и решение проблем — в файле **ЗАПУСК.md**.

## Оповещения в Telegram

1. Создайте бота через [@BotFather](https://t.me/BotFather), получите токен.
2. Напишите боту `/start`. Узнайте свой Chat ID через [@userinfobot](https://t.me/userinfobot).
3. В папке `server` создайте файл `.env`:
   ```
   TELEGRAM_BOT_TOKEN=123456:ABCdef...
   ```
4. Запустите и клиент, и сервер (`npm run dev`). Во вкладке «Оповещения» включите телеграм-уведомления и введите Chat ID. Список оповещений синхронизируется с сервером; сервер раз в минуту проверяет условия и отправляет сообщения в Telegram.

## Возможности

- **Скринер** — сетка графиков по умолчанию по волатильности и объёму (топ роста / по объёму). Редактор рабочего пространства: фильтры (объём, изменение цены, волатильность, корреляция с BTC), сортировка, формации, таймфреймы 1m, 5m, 15m, 1h, 4h, 1d.
- **Монеты** — список фьючерсов Bybit с объёмом, изменением цены, волатильностью; большой график и тех. данные по выбранной монете.
- **Карта плотностей** — стакан (bid/ask) по выбранному символу, визуализация крупных плотностей для отторговки.
- **Оповещения** — типы: пересечение цены, плотности, изменение цены, корреляция с BTC, волатильность, всплеск объемов, листинги, фандинг, открытый интерес. Сохранение в браузере и синхронизация с сервером для отправки в Telegram.

Данные (тикеры, свечи, стакан, фандинг) берутся только с Bybit (публичное API v5).
